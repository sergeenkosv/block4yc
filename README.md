# block4YC

## (in English)

The batch file `block4yc-ext.bat` is intended to start and configure Windows Defender firewall
in such a way that any external resouces except ext.contest.yandex.{ru,com} and what it is depended on will not be available.

The batch file is generating unblocking batch file during execution.

The unblocking batch file will revert changes. To do so it need *.wfw file generated by block4yc.

The batch file `block4yc.bat` has a purpose similar to `block4yc-ext.bat` with one exception. It allows access not only to ext.contest.yandex.{ru,com}, but also to official.contest.yandex.{ru,com} and contest.yandex.{ru,com}

## (на русском)

Пакетный файл `block4yc-ext.bat` предназначен для запуска и настройки брандмауэра Windows Defender таким способом что никакие внешние ресурсы за исключением ext.contest.yandex.{ru,com} и тех, от которых он зависит, не будут доступны.

Этот файл генерирует пакетный файл для снятия блокировки в процессе своего выполнения.

Разблокирующий пакетный файл отменяет изменения. Для этого ему нужен файл *.wfw, сгенерированный block4yc.

Пакетный файл `block4yc.bat` имеет цель схожую с `block4yc-ext.bat` с единственным исключением. Он разрешает доступ не только к ext.contest.yandex.{ru,com}, но также к official.contest.yandex.{ru,com} и contest.yandex.{ru,com}
